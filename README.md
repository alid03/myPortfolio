Portfolio Website - Ali Dabdoub
This is a personal portfolio website designed to showcase projects, skills, and professional information.

Project Overview
This website is built to provide a digital resume and portfolio for me, Ali Dabdoub, a Software Engineering student. It includes sections for a personal introduction, a downloadable resume, showcased projects, and contact information. This is my first
project, my first repository and a lot more other things, and I hope to improve more in the future. This project has things to work on, but I am proud of my work, while still pushing to improve it.

Features
Theme Selector: Users can choose between multiple color themes: Light, Blue, Green, and Purple.
Responsive Design: The website adapts to different screen sizes.
Project Showcase: A placeholder for showcasing projects (with plans to update in the future).
Contact Form: A basic form for users to get in touch.
Live Website
You can view the website live at: http://alidabdoub.s3-website.us-east-2.amazonaws.com/

Files and Structure
HTML Files

index.html: The main file that structures the content.
post.html: A template for individual project posts (currently placeholders).
CSS Files

default.css: The default (light) theme.
blue.css, green.css, purple.css: Alternate color themes for the website.
bucket-policy.txt: A sample S3 bucket policy for allowing public read access (for reference purposes).
JavaScript File

script.js: Handles theme switching and saves the user’s theme preference in local storage.
Assets

Images used for profile pictures, project placeholders, and social links.
Getting Started
Prerequisites
Any web browser.
Running Locally
Download or clone the repository.
Open index.html in your browser to view the website.
Customizing the Website
You can easily modify the theme colors by editing the CSS variables in the respective .css files.
Update the index.html content to add new projects or update contact information.
Future Improvements
Replace placeholder project images and descriptions with actual projects.
Add more interactive elements, like project demos or links to live applications.
